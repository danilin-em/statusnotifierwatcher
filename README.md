# StatusNotifierWatcher

Get apps from tray

## Usage:

```bash

# 0: Init VENV
python3 -m virtualenv venv && source venv/bin/activate

# 0.1: Install requirements
pip install -r requirements.txt

# 1: Start Watcher
python3 statusnotifierwatcher.py

# 1.1 Start Proxy
xembedsniproxy

# Now see apps in tray
python3 myapps.py

```
